# SUGA-UNet: Enhanced Deep Learning for Liver Tumor Segmentation

## Project Overview

This project focuses on automatic liver tumor segmentation from CT scan images using an enhanced U-Net based deep learning architecture.

The proposed model integrates attention mechanisms, multi-scale feature extraction, and uncertainty-guided learning to improve segmentation performance.

---

## Dataset

AndrewMVD Liver Tumor Segmentation Dataset
